Astro Anti Link Privacy Policy

Last updated: June 21, 2026

Astro Anti Link is a Discord moderation bot designed to help server staff block unwanted links and protect communities.

Information the Bot Can Access

To detect and remove blocked links, the bot may access message content in servers where it is installed. The bot may also access member, role, nickname, permission, channel, and server information needed to check permissions, run slash commands, and apply moderation settings.

Information Stored

The bot stores basic server configuration such as whether anti-link protection is enabled, advanced mode status, warning settings, blocked domain lists, and the selected log channel ID.

The bot does not intentionally store private message content in its files. If a log channel is configured, deleted-link reports may be posted in that server's chosen log channel for staff review.

How Information Is Used

  • To detect and delete blocked links.
  • To allow server administrators to configure anti-link settings.
  • To send moderation warnings and optional log messages.
  • To prevent abuse, spam, and unwanted invite/link posting.

Data Sharing

The bot does not sell personal data. The bot does not share stored configuration data with advertisers or unrelated third parties.

Data Removal

Server administrators may remove the bot from their server at any time. Stored server configuration can be reset by using the bot's reset settings command or by contacting the bot owner/operator.

Security

Reasonable steps are taken to protect bot configuration data. No online service can guarantee perfect security.

Contact

For privacy questions or data removal requests, contact the bot owner/operator through the Discord server where the bot is installed.
