# Astro Anti Link Policy Pages

These files are ready-to-host Terms of Service and Privacy Policy pages for your Discord bot.

## What to put in Discord Developer Portal

Leave these blank unless you are building an advanced HTTP endpoint / linked-role app:

- Interactions Endpoint URL: leave blank
- Linked Roles Verification URL: leave blank

Use your hosted page URLs for these:

- Terms of Service URL: https://YOUR-DOMAIN.com/terms-of-service.html
- Privacy Policy URL: https://YOUR-DOMAIN.com/privacy-policy.html

## Simple hosting options

Upload these files to GitHub Pages, Netlify, Vercel, your website, or any public HTTPS hosting.
Discord needs a public HTTPS URL, not a local file path.
